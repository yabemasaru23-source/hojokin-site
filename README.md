# 三社 補助金・助成金 管理サイト

## 開き方

`index.html` をブラウザで開くと、ローカルでそのまま使えます。

## GitHub Pagesで公開する場合

1. GitHubで新しいリポジトリを作成する
2. このフォルダをそのリポジトリにアップロードする
3. GitHubの `Settings` → `Pages` を開く
4. `Build and deployment` の `Source` を `Deploy from a branch` にする
5. `Branch` を `main`、フォルダを `/root` にして保存する
6. 数分後に表示されるURLを開く

公開URLを知っている人は閲覧できるため、口座情報、パスワード、ログイン情報、詳細な個人情報は載せないでください。

## 共有のしかた

- このフォルダを共有ドライブやOneDrive共有フォルダに置く
- メンバーは `index.html` を開く
- 進捗を共有したい人が「進捗を出力」でJSONを保存する
- 他の人は「進捗を取込」でそのJSONを読み込む

完全に同時編集できる共有サイトにする場合は、次のどちらかが必要です。

- GoogleスプレッドシートやNotionなどを進捗データの置き場にする
- 小さなサーバーやデータベースを用意して、全員の進捗を保存する

## 更新するとき

補助金情報は `data.js` にまとまっています。

- 会社情報: `companies`
- 補助金・助成金: `grants`
- 準備チェックリスト: `checklist`
- 締切カレンダー: `milestones`

申請前には、必ず各制度の公式URLで募集要項、対象者、申請期間、対象経費を確認してください。
